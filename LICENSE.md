## 🔧 Common Issues & Fixes

### ❌ "Injection failed" on Windows
**Cause:** Antivirus flagged and removed a Seliware core file.  
**Fix:** Add the Seliware folder to your antivirus exclusions and reinstall. You can verify file integrity against the **seliware github** repository.

### ❌ UI theme resets after every launch
**Cause:** Theme config file lacks write permissions.  
**Fix:** Right-click the Seliware folder → Properties → Security → grant your user account **Full Control**.

### ❌ Executor won't open on macOS
**Cause:** Gatekeeper blocking the app after a macOS update.  
**Fix:** Go to **System Settings → Privacy & Security** and click "Open Anyway" — this resets after system updates.

### ❌ Scripts run but produce no output
**Cause:** Console panel minimized or hidden in the **seliware ui**.  
**Fix:** Click the **Console** tab in the Seliware interface — all `print()` output is captured there.

### ❌ Seliware closes immediately on launch
**Cause:** Missing .NET 6.0 runtime.  
**Fix:** Install the [.NET 6.0 Runtime](https://seliware-doc.github.io/seliware/) from Microsoft, then relaunch Seliware.

### ❌ Custom UI changes not saving
**Cause:** Seliware installed in a write-protected directory (e.g., Program Files).  
**Fix:** Move the Seliware folder to a user-writable location like `C:\Users\YourName\Seliware`.
